# Java-Swing-Practice-
A collection of Java Swing programs created while learning desktop application development using Java Swing.
This repository serves as a hands-on learning journal where I practice fundamental Swing components, layout managers, and event handling concepts.

## Topics Covered

### Core Components
- JFrame
- JLabel
- JButton
- JPanel
- JTextField
- JComboBox
- JCheckBox
- JRadioButton
- JOptionPane
- JSlider
- JTable
- JProgressBar
- JMenuBar

### Layout Managers
- FlowLayout
- BorderLayout
- GridLayout
- JLayeredPane

### Event Handling
- ActionListener
- KeyListener
- MouseListener
- ChangeListener

### Features Practiced
- Creating and customizing application windows
- Adding text, images, and icons
- Handling button clicks and keyboard events
- Working with different layout managers
- Taking user input through various Swing components
- Displaying dialog boxes
- Grouping radio buttons
- Creating menus and menu bars
- Building tables
- Implementing sliders and progress bars
- Managing layered components
- Customizing colors, fonts, and window properties

## Technologies Used

- Java
- Java Swing
- IntelliJ IDEA
- Git & GitHub
